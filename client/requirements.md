## Packages
browser-image-compression | Robust client-side image compression library that runs entirely in the browser using Web Workers
react-dropzone | High quality drag and drop file upload component for React
framer-motion | Smooth animations for UI state transitions
lucide-react | Professional iconography

## Notes
- This is a 100% client-side application. No backend API calls are made for image processing.
- Uses `browser-image-compression` to handle resizing and quality reduction in a Web Worker to keep the UI thread responsive.
- AdSense placeholders are implemented as responsive div blocks that match standard AdSense unit sizes.
